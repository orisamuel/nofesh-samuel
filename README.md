# 🌅 נופש סמואל — "איזה יום היה לי סמואל"

אתר לניהול הנופש המשפחתי: לו״ז, השתבצויות (ארוחות/פעילויות/כלים), רשימת ציוד, משחקונים עם נקודות וטבלת מובילים, וקיר שיתוף — הכל משותף לכל המשפחה בזמן אמת.

## המבנה (Stack)
- **Frontend:** HTML/CSS/JS טהור, בלי build. עמוד אחד (SPA), RTL, מותאם מובייל, מצב בהיר/כהה.
- **Backend:** Google Apps Script (קובץ `.gs` יחיד).
- **מסד נתונים:** Google Sheets (אתם עורכים תוכן ישירות בגיליון).
- **אחסון:** GitHub Pages (חינם) — או כל אירוח סטטי.

---

## 🚀 הפעלה — פעם אחת (כ-15 דקות)

### שלב 1 — יוצרים את הגיליון (מסד הנתונים)
1. נכנסים ל-https://sheets.google.com ← גיליון ריק חדש.
2. נותנים שם, למשל **"נופש סמואל — DB"**.
3. מהכתובת `https://docs.google.com/spreadsheets/d/<SHEET_ID>/edit` מעתיקים את ה-**SHEET_ID** (המחרוזת הארוכה בין `/d/` ל-`/edit`).

### שלב 2 — יוצרים את ה-Backend (Apps Script)
1. בתוך הגיליון: תפריט **Extensions ← Apps Script**. נפתח טאב חדש.
2. מוחקים את `function myFunction() {}` שקיים.
3. פותחים את הקובץ [`appscript.gs`](appscript.gs), מעתיקים את **כל** התוכן ומדביקים בעורך.
4. בשורה הראשונה מחליפים `PASTE_YOUR_SHEET_ID_HERE` ב-SHEET_ID משלב 1.
5. שומרים (💾 או Ctrl+S).

### שלב 3 — הרשאה ראשונית
1. בעורך, בוחרים בתפריט הפונקציות את `getSettings` ← לוחצים ▶ **Run**.
2. Google יזהיר "This app isn't verified" ← **Advanced ← Go to project (unsafe) ← Allow**.
3. זה מאשר לסקריפט לקרוא/לכתוב **בגיליון שלכם בלבד** (פעם אחת).

### שלב 4 — פורסים כ-Web App
1. **Deploy ← New deployment** ← גלגל השיניים ⚙ ← **Web app**.
2. מגדירים:
   - Description: `v1`
   - **Execute as: Me**
   - **Who has access: Anyone** ← חשוב! בלי זה האתר לא יוכל לקרוא.
3. **Deploy** ← מעתיקים את ה-**Web app URL** (מסתיים ב-`/exec`).

### שלב 5 — מחברים את ה-Frontend
פותחים את [`config.js`](config.js) ומעדכנים:
```js
SCRIPT_URL: 'https://script.google.com/macros/s/XXXX/exec',   // מה שהעתקתם בשלב 4
SHEETS_URL: 'https://docs.google.com/spreadsheets/d/<SHEET_ID>/edit',
```
זהו — פותחים את `index.html` בדפדפן והאתר עובד עם נתונים אמיתיים.

> ✨ **בונוס:** בפעם הראשונה שהאתר ניגש לגיליון, הוא **נזרע אוטומטית** בכל התוכן מהקבוצה — הלו״ז, הארוחות, רשימת הכלים והציוד, הפעילויות וגם חידון "מי אמר את זה?". אין צורך להקליד כלום ידנית.

### שלב 6 — מעלים לאוויר (GitHub Pages, חינם)
1. מעלים את תיקיית הפרויקט ל-repo חדש ב-GitHub.
2. ב-repo: **Settings ← Pages** ← Source: ענף `main`, תיקייה `/root` ← Save.
3. אחרי דקה האתר באוויר בכתובת `https://<user>.github.io/<repo>/` — שולחים לכל המשפחה בוואטסאפ 🎉

### שלב 7 (מומלץ) — שומרים את השרת "חם"
Apps Script "נרדם" אחרי כמה דקות והקריאה הבאה איטית (3-5 שנ'). למניעה:
בעורך ← 🕒 **Triggers** ← Add Trigger ← Function: `keepWarm`, Time-driven, Every 10 minutes.

## ✏️ איך עורכים תוכן

הכול נערך **ישירות בגיליון** (לשוניות בתחתית). אחרי עריכה בגיליון — פשוט מרעננים את האתר, **בלי לפרוס מחדש**.

| לשונית | למה זה |
|--------|--------|
| `settings` | תאריך הנופש, מיקום, קישור Waze, סלוגן |
| `schedule` | הלו״ז (יום, שעה, כותרת, סוג, פירוט) |
| `assignments` | פריטים להשתבצות (ארוחות/פעילויות/כלים) + כמה צריך (`needed`) |
| `packing` | רשימת "מה להביא" |
| `quiz` | שאלות החידון (`correct` = מספר התשובה הנכונה 1-4) |
| `users` / `claims` / `scores` / `posts` | מתמלאות אוטומטית מהאתר — לא לגעת בד"כ |

### 📅 לעדכן את תאריך הנופש (חשוב!)
בלשונית `settings`, בשורה `vacationStart`, מזינים תאריך בפורמט `dd/MM/yyyy HH:mm`
(כרגע זרוע `26/07/2026 16:00` כברירת מחדל). זה מפעיל את הספירה לאחור.

### 📷 להוסיף תמונות משפחתיות למשחק הזיכרון
1. שמים תמונות בתיקייה `תמונות/` (לפחות 6).
2. ב-[`config.js`](config.js) מעדכנים את `MEMORY_IMAGES` עם שמות הקבצים:
   ```js
   MEMORY_IMAGES: ['saba.jpg','savta.jpg','dod.jpg','ima.jpg','kelev.jpg','bayit.jpg'],
   ```
   (אם ריק — המשחק משתמש באימוג'ים.)

### ❓ להוסיף שאלות לחידון
מוסיפים שורה בלשונית `quiz`: קטגוריה, שאלה, 4 תשובות, ו-`correct` = מספר התשובה הנכונה (1-4).
הקטגוריה `מי אמר את זה?` נועדה לציטוטים מהקבוצה 😄

## 🔧 עדכון ה-Backend
⚠️ **בכל שינוי ב-`appscript.gs` חובה לפרוס מחדש** (Apps Script לא מתעדכן לבד):
**Deploy ← Manage deployments ← ✏ (עריכת הפריסה) ← Version: New version ← Deploy.**
ה-URL נשאר אותו דבר — אין צורך לשנות ב-config.

## 👀 תצוגה מקדימה מקומית (לא חובה)
לפני הפריסה אפשר לראות את האתר מקומית:
```bash
python -m http.server 8123
```
ואז נכנסים ל-http://localhost:8123 . (בלי גיליון מחובר — "מצב תצוגה מקדימה": העיצוב עובד, הנתונים לא נשמרים.)

## 🔒 הערת אבטחה
זו פלטפורמת "אמון" למשפחה. הקוד האישי (PIN) הוא הגנה קלה בלבד ונשמר כטקסט בגיליון.
כל מי שיש לו את ה-`SCRIPT_URL` יכול לקרוא/לכתוב. **לא לאחסן כאן מידע רגיש** (תעודות זהות, כספים וכו').

## 📁 מבנה הקבצים
```
index.html      · שלד האתר
styles.css      · מערכת העיצוב ("גלויית קיץ מהגליל")
config.js       · הגדרות — כאן מדביקים את SCRIPT_URL
utils.js        · כלים משותפים (apiCall, טוסט, ערכת נושא)
app.js          · ניווט, פרופילים, בית, לו״ז, השתבצויות, מה להביא, קיר
games.js        · משחק זיכרון, חידון, טבלת מובילים
appscript.gs    · כל ה-Backend (מדביקים ב-Apps Script)
תמונות/         · תמונות משפחתיות (למשחק הזיכרון ולעיצוב)
```
